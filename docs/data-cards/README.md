# Data Cards Directory

This directory registers the datasets and reference tables utilized to train classifiers, validate predictions, or establish baseline risk values.

## 1. Active Data References
- **FINDRISC Baseline (Finnish Diabetes Study)**: Statistical table mapping scores to risk tiers.
- **Framingham Cohort Statistics**: Baseline coefficients for risk regression curves.
- **Food Nutrition Reference Table**: Ingredients database utilized for personalized food scanning.

## 2. Licensing and Clinical Consent
- Use only public domain clinical guidelines.
- Synthetic profiles used in test suites must be labeled as synthetic.
